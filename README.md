# 🔐 Encryption and Decryption Tool (Python)

## 📌 Project Overview

This project is a simple **Encryption and Decryption system** built using Python. It allows users to securely convert plain text into encrypted format and decrypt it back to the original message.

The main objective of this project is to understand basic concepts of **data security, cryptography, and text transformation techniques**.

---

## 🎯 Key Features

* 🔒 Encrypt plain text into secure format
* 🔓 Decrypt encrypted text back to original
* ⚡ Fast and lightweight implementation
* 🧠 Demonstrates basic cryptographic logic
* 💻 Easy to use Python script

---

## 🛠️ Technologies Used

* **Python** – Core programming
* Basic string manipulation & logic building

---

## 📁 Project Structure

* `Secrete.py` → Main script for encryption and decryption
* `README.md` → Project documentation

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/rutik-bhendarkar/Encryption_And_Decryption.git
   ```
2. Navigate to the project folder:

   ```bash
   cd Encryption_And_Decryption
   ```
3. Run the Python script:

   ```bash
   python Secrete.py
   ```
